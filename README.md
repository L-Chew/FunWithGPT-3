# FunWithGPT-3

## Description
An application that sends plain text prompts to the OpenAI API and displays the results in a list.

## Obtain API Key
- Go to https://beta.openai.com/signup
- Enter your email address and password
- Verify your email address
- Verify your phone number by entering the code that is sent to you via SMS

## Available Scripts

In the project directory, you can run:

### `npm install`

Install dependencies for the project.

### `config.js`

Duplicate `exampleConfig.js` in the root directory.\
Rename to `config.js`.\
Edit `config.js` file with obtained OpenAI token.

### `npm start`

Start Server:\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm run build`

It correctly bundles React in production mode.
